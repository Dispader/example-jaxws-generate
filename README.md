# JAX-WS wsimport example

A simple project testing JAX-WS's `wsimport`.

## usage

* `./gradlew test`

## Eclipse integration

* bash
  * `./gradlew eclipse`
* Eclipse
  * `File` -> `Import...`
  * `Existing Projects into Workspace` -> `Next`
  * `Select root directory` -> `Browse` -> <project directory>
  * `Finish`
